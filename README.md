# Abitur BW 2028 – Notenrechner

Kurs- & Notenrechner für die gymnasiale Qualifikationsphase in Baden-Württemberg (Abitur 2028).

## Features

- Leistungsfächer wählen mit Live-Validierung
- Block I Berechnung (40 Kurse, Doppelgewichtung)
- Block II Berechnung (schriftlich/mündlich, Zusatzmündliche)
- Abiturnote live berechnet
- Bestanden/Nicht-bestanden Prüfung aller Bedingungen

## Datenschutz

Alle Daten bleiben lokal im Browser. Kein Server, keine Datenbank, kein Tracking.

## Hosting

Statische HTML-Datei – läuft auf GitHub Pages oder jedem Webserver.

## Grundlage

[Leitfaden für die gymnasiale Oberstufe – Abitur 2028 (KM BW)](https://km.baden-wuerttemberg.de/fileadmin/redaktion/m-km/intern/PDF/Publikationen/Gymnasium/2025-12-16-Leitfaden-ABI-2028.pdf)
